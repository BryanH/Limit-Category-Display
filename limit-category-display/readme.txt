=== Limit Category Display ===
Contributors:
Tags: post, categories
Requires at least: 3.0.0
Tested up to: 3.0.1
Stable tag: git://github.com/BryanH/Limit-Category-Display.git

Limits the number of categories displayed on all posts on the index page.

== Installation ==

1. Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation

2. Activate the Plugin from Plugins page.

3. Go to the options menu, select the 'Limit Category' option and enter the maximum number of categories to display.
