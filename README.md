Limit Category Display
======================

Limit Category Display is a Wordpress plugin that affects the display of a post on index pages.


## Installing
* Copy the directory into your `wp-content/plugin` directory.
* Go into your Wordpress admin screen and activate the plugin.

## Using
* Go to the blog's settings menu and click on the 'Limit Category' options.
* Add the maximum number of categories to display.
* Index page posts will show no more than the number of categories you specified.

Note - a post can have many more categories than what are displayed.
